    #ClipCloud

    Have you ever pasted a screenshot in to Microsoft Paint to email it to a friend? You might like this.

    ClipCloud is an attempt to consolidate all the types of data you want to share, and all the places you want to send it to in to one program.

    Currently it supports uploading files, taking screenshots and sharing text snippets. You can upload to Dropbox.

    In the future the plan is to support more kinds of data, and more places to host it.

    Mostly written in Python, only tested with Python 2.7 on Windows, but the plan is to support all platforms. Windows is a priority because we don't have similar services like CloudApp already.

    While in development it wont work with the Dropbox API unless you make your own app and authorise it with a new API key, but feel free to experiment with it anyway.
